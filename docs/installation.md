# Installation Guide (Beginner-Friendly)

## What this step does

Installation prepares ThoughtTagger so it can run on your computer.

## Prerequisites

- **Node.js 20+**
- **npm 10+** (usually installed together with Node.js)

## Where to run commands

Run commands in a **Terminal** opened at the repository root folder.

- Windows: PowerShell / Command Prompt
- macOS: Terminal
- Linux: Terminal

## Install dependencies

```bash
npm install
```

## Verify installation

```bash
npm run build
npm test
```

If both commands finish successfully, your installation is ready.
