# ThoughtTagger Studio (V1 Product UI)

Researcher-facing and annotator-facing runnable UI.

## Run locally

```bash
npm install
npm run build
npm run studio
```

Open: `http://localhost:4173`

## What you can do now

1. Paste `study.spec.json` and `dataset.jsonl`.
2. Create a study runtime.
3. Start an annotator session with `annotator_id`.
4. Navigate units and answer rubric questions.
5. Autosave locally in browser (`localStorage`).
6. Save session to server file (`apps/studio/.localdata`).
7. Export session JSON.
